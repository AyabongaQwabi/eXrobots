# eXrobots

Basic led projects

## Required

* Arduino
* Breadboard
* 330 ohm Resistor x 1
* Jumper Wires >= 6
* Trimpot  (Trimmer Potentiometer x 1)


## led_dim.js Setup Diagram

[How to Setup led_dim Diagram ](https://github.com/denvereezy/eXrobots/wiki/led_dim.js)


Run `node led_dim.js` or `nodemon led_dim.js`
